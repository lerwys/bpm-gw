RESERVED[4:0] PRST PWR_GOOD MMCM_LOCKED
  • MMCM_LOCKED [read-only]: MMCM locked status
    ADC Clock Chain MMCM locked status.
    read 0: not locked
    read 1: locked
  • PWR_GOOD [read-only]: FMC power good status
    FMC power good status.
    read 0: power not good
    read 1: power good
  • PRST [read-only]: FMC board present status
    FMC board present status.
    0: not present
    1: present
  • RESERVED [read-only]: Reserved
    Ignore on write, read as 0's

  • DIR [read/write]: Direction
    Trigger direction
    0 - output
    1 - input
  • TERM [read/write]: Termination Control
    Trigger termination 50 ohm
    0 - disable
    1 - enable
  • TRIG_VAL [read/write]: Trigger Value
    Trigger value (when used in output mode)
  • RESERVED [read-only]: Reserved
    Ignore on read, write with 0's.

  • TEST_DATA_EN [read/write]: Enable test data
    Write the address counter value instead of ADC data to Wishbone Streaming Interface
  • LED1 [read/write]: Led 1
    FMC LED1 (blue) - configuration in progress indicator
    0 - LED off - LED on
  • LED2 [read/write]: Led 2
    FMC LED2 (red) - data acquisition in progress indicator
    0 - LED off - LED on
  • LED3 [read/write]: Led 3
    FMC LED3 (green) - trigger status indicator
    0 - LED off - LED on
  • MMCM_RST [read/write]: MMCM reset
    write 1: reset MMCM.
    write 0: no effect
  • RESERVED [read-only]: Reserved
    Ignore on read, write with 0's.

3.2. Global ADC Control register

HW prefix: wb_fmc_250m_4ch_csr_adc_ctl
HW address: 0x1
C prefix: ADC_CTL
C offset: 0x4

Global ADC control register.

23 22 21 20 19 18 17 16
RESERVED[18:11]

RESERVED[10:3]
7 6 5 4 3 2 1 0
RESERVED[2:0] SLEEP_ADCS RST_DIV_ADCS RST_ADCS UPDATE_DATA_DLY UPDATE_CLK_DLY
  • UPDATE_CLK_DLY [read/write]: Reset/Update ADC clock chains delay
    write 1: reset/update clock chains to its default
    values or from the values in the delay registers.
    write 0: no effect
  • UPDATE_DATA_DLY [read/write]: Reset/Update ADC data chains delay
    write 1: reset/update data chains to its default
    values or from the values in the delay registers.
    write 0: no effect
  • RST_ADCS [read/write]: Reset ADCs
    Reset and recalibrate ADCs
    write 0: no reset
    write 1: enable ADC reset
  • RST_DIV_ADCS [read/write]: Reset Div ADCs
    Reset for synchronization between multiple ADCs
    write 0: no reset
    write 1: enable ADC div reset
  • SLEEP_ADCS [read/write]: Sleep ADCs
    Sleep for ADCs
    write 0: normal operation
    write 1: enable sleep mode
  • RESERVED [read-only]: Reserved
    Ignore on write, read as 0's
